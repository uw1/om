# 数据库
个人数据可以用json，爬来的数据再用json有点问题吧
mongodb等又有迁移|同步的问题
sql有语言|api的问题
nosql其实也有过于啰嗦的问题，但可以自己封装简化
mongodb没法一直用
真正的nosql应该是图数据，以entity为主，而不是以表为主
某一种表也是entity
entity就是个id，想怎么存就怎么存
而不是被限制成一个schema
索引本质是一种关系entity
