# GB2312 汉字编码字符集
[GB 2312 - 维基百科，自由的百科全书](https://zh.wikipedia.org/wiki/GB_2312)
共收录6763个汉字，一级汉字3755个，二级汉字3008个；同时收录了包括拉丁字母、希腊字母、日文平假名及片假名字母、俄语西里尔字母在内的682个字符。
1980年
