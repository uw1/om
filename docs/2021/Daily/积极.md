# 积极思维
积极思维，不是所谓的乐观向上。而是去专注于优点，去专注于什么，怎么样把一件事情做好，而不是避免怎么样。做不好，去抓住人的优点，而不是缺乏。
追求"完美"，导致连开始都困难
#
盯着事物的缺点看没有任何意义，只会给自己找不快。应该跟着事物的优点看，因为这个是没有优点，那就不去接受它，忽略它跳过它就OK了，没有必要去浪费其他的时间精力去看盯着一个事物的缺点去批判。
要怎么利用优点
