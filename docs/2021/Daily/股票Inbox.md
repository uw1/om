# 股市的普世意义
股市并不只是赌市，炒股也不只是不劳而获。
股市是一种真正且有用的民主制度，以所有上市公司为候选者。钱就是选票，错误的投票会受到惩罚。
反观总统选举这种搞笑的民主制度，愚蠢的投票者没有一点惩罚，错误的投票让整个国家(乃至世界)的所有人买单。
而在股市中，聪明的投票者和有价值的公司会一起越来越有钱，国家也随之变得更好。
一个人的投资微不足道，但所有人的(正确)投资则能让国家变好。
