# 书籍的优势是什么？
更系统。逻辑更完整。
更深入。一些内容只有在书中可以看到。
量大管饱。找到了之后可以看很久。不需要一直找。
# 书籍的劣势是什么？
更加繁杂。什么细枝末节都讲。
知识相对老旧，不适合最新的信息。
