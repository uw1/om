# insight
遇到问题并不需要去检测解决，先停下来。先思考，先休息，然后再去解决。不应该追求当下的效率，而是应该追求未来长远的效率。
grow 思维太消极，过于重视不好的东西。总是被各种问题卡住。但实际上很多问题并不需要去解决。重点并不是问题。而是过程和结果。价值和系统
