# 原则
化想象为己用，而不是被利用
首先是学会关掉/忘掉不好的想象
不需要真正的发生，只要能想象出来就行了
不爽/负面情绪其实也是来源于不好的想象(无意识的)
做了错事不代表一定是错的结果，只有再加上错的反应才会导致错的结果
