# 减少而非消除
极简是不断减少，而不是消除，消除思维是错的
# 更少而非更多
遇到好的东西，想要更多是本能，但越多质量越差
	为什么要把东西吃完
浪费时间不只是你把时间花在没用的事上，还包括不够重要不够好的事
# 数量与质量成反比
一个东西数量越多质量越低，越不值得花费资源在上面
只有数量最少时，质量是最高的，最值得花费的
