# 情绪
处理负面事物的最好方法就是忽略第一时间忽略他们。而不是去想他们哪里不好。怎样不好？怎么这么不好？
立刻把注意力转移到下一个事物上面。
如果可以避免的话，下次应该怎么样避免。

大部分的想法。大脑里的东西其实都是不重要的。
